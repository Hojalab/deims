Configure information for EML output
--------------------------------------------------------------------------------
This module works with the module views_bonus_eml to provide information on 
intellectualRights, language, metadataProvider, and publisher, used in all EML 
files for the site.

use path http://your.domain.edu/eml_config to enter the information
--------------------------------------------------------------------------------
Some formatting for paragraphs in the intellectual rights can be achieved by 
entering </literalLayout></para><para><literalLayout> at paragraph breaks.
The initial and ending tags are provided by the EML export program, i.e. should
not be entered into the text.

e.g.
first paragraph</literalLayout></para><para><literalLayout>second paragraph
</literalLayout></para><para><literalLayout>last paragraph
